Overview
========

Lesson contents:

1. :doc:`Geometric operations: Overlay analysis <geometric-operations>`
2. :doc:`Data classification <reclassify>`
3. :doc:`Exercise 4 <ex-4>`

Learning goals
--------------

AFter this class, you should be able to

- Conduct overlay analysis
- Aggregate geometries
- Simplify geometries
- Classify data based on different criteria using common classifiers, and custom classifiers
